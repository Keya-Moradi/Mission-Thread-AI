# docs/assets

Reserved location for the project banner and later screenshots.

## Banner

`README.md` references a banner image at:

```
docs/assets/missionthread-ai-banner.png
```

That file has not been placed in the repository yet — it was generated in a
separate tool session and this assistant has no access to that binary file,
so it was not fabricated here. To complete the banner:

1. Save the generated image as `docs/assets/missionthread-ai-banner.png`
   (roughly 16:9 works well for both the README and GitHub's social preview).
2. It will then render automatically at the top of `README.md`.
3. Optionally upload the same image under GitHub → Settings → General →
   Social preview → Edit, and confirm the title stays legible after GitHub's
   crop.
