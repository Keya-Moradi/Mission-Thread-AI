# docs/assets

Reserved location for the project banner and later screenshots.

## Banner

`README.md` renders `docs/assets/missionthread-ai-banner.png` (1280×640) at
the top of the page. To update it, replace that file — no other changes are
needed, since the README references it by a stable, repository-relative
path. The same image can optionally be uploaded under GitHub → Settings →
General → Social preview → Edit; confirm the title stays legible after
GitHub's crop.

## Screenshots

Phase 8 (`docs/SPEC.md` §19) adds screenshots of the running app here once
the workflow UI is built out past the Phase 1 shell.
